店铺管理

网站系统店铺索引与管理

操作提示

  • 1. 平台在此处统一管理自营店铺，可以新增、编辑、删除平台自营店铺。
  • 2. 可以设置未绑定全部商品类目的平台自营店铺的经营类目。
  • 3. 已经发布商品的自营店铺不能被删除。
  • 4. 删除平台自营店铺将会同时删除店铺的相关图片以及相关商家中心账户，请谨慎操作！
